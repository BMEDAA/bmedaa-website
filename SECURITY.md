# 🔒 安全說明

## 重要提醒

此專案包含真實的校友會員資料，請務必保持倉庫為**私有狀態**。

## 敏感資料

### 已受保護的檔案
- `account info.csv` - 會員帳號密碼
- `frontend/src/data/mockData.ts` - 真實會員資料
- 環境變數檔案

### 安全措施
1. ✅ 使用私有GitHub倉庫
2. ✅ `.gitignore` 配置完善
3. ✅ 生產環境隱藏敏感資訊
4. ✅ 測試帳號僅在開發環境顯示

- 環境變數將在Vercel控制台設定
